Programming

Programming is the process of creating instructions that a computer can understand and execute. These instructions are written in programming languages like Python, C, Java, or JavaScript.

Key Concepts:

  • Program: A set of instructions to perform a specific task.
  • Programming Language: Language used to write programs.
  • Compiler/Interpreter: Converts code into machine-readable instructions.
  • Syntax: Rules for writing a valid program.
Example:
Python program to add two numbers:
a = 5
b = 3
sum = a + b
print("Sum =", sum)

Algorithm

An algorithm is a step-by-step procedure to solve a problem. It is independent of any programming language.

Characteristics of a Good Algorithm:

  • Clear and unambiguous.
  • Finite steps (must terminate).
  • Input(s) specified and output(s) defined.
  • Efficient in time and space.

Example


Algorithm to find the sum of two numbers:
  1. Start
  2. Read number1 and number2
  3. Sum = number1 + number2
  4. Print Sum
  5. End

Types of Algorithms

  • Sorting Algorithm: Arranges data in a specific order (ascending/descending).
  • Searching Algorithm: Finds a specific element in a dataset.
  • Recursive Algorithm: Algorithm that calls itself.
  • Divide and Conquer: Breaks problem into smaller subproblems.

4. Programming Flow

Writing a program usually follows these steps:

  1. Understand the problem.
  2. Design an algorithm.
  3. Write code in chosen programming language.
  4. Test and debug the program.
  5. Run the program to get output.

Examples

Example 1: Find the largest of two numbers (Python)
a = 10
b = 20
if a > b:
  print("a is larger")
else:
  print("b is larger")

Example:

Check if a number is even or odd (Python)
n = 7
if n % 2 == 0:
  print("Even")
else:
  print("Odd")

Example

: Sum of first n natural numbers (Algorithm)
  1. Start
  2. Read n
  3. sum = 0
  4. For i = 1 to n do
  5.   sum = sum + i
  6. Print sum
  7. End

Example

: Bubble Sort Algorithm

Sort an array of numbers in ascending order:

  1. Start
  2. Read array of n elements
  3. Repeat steps for i = 0 to n-1
  4.   Repeat for j = 0 to n-i-1
  5.     If array[j] > array[j+1], swap them
  6. Print sorted array
  7. End
