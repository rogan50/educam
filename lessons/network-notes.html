Computer Network

Meaning of Computer Network

A computer network is a collection of two or more computers and other devices connected together so that they can share rerources, such as files, printers, and internet connections.

Definition

A computer network is a system where computers are linked together to communicate and share information.

Examples

  • Computers connected in a school's computer lab.
  • Mobile phones connected via Wi-Fi.
  • ATMs linked to a central banking system.

Basic Terms

  • Node: Any device connected to a network (computers, printer, router, etc.)
  • Server: A computer that provides services or resources to others on the network
  • Client: A computer that receives services from the server
  • Network Interface Card (NIC): Hardware that connects a computer to the network
  • Transmission Media: The communication path (cables or wireless signals) used to transmit data
  • Protocol: Rules that guid how data is sent and received in a network

Types of Computer Networks

Personal Area Network (PAN)

  • Very small network used by one person.
  • Used to connect personal devices like phones, tablets, and laptops

Examples: Bluetooth, Infrared, Hotspot.

Local Area Network (LAN)

  • Covers a small area such as a school, cyber cafe, or office.
  • Owned by one organization.
  • Uses cables (Ethernet) or Wi-Fi

Example: A school lab network where all computers share one printer.

Metropolitan Area Network (MAN)

  • Covers a larger area, such as a town or city
  • Often used by Companies or service providers to connect multiple buildings

Example: University campuses or citywide Internet Service Provider networks.

Wide Area Network (WAN)

  • Cover a very large geographical area countries or continents.
  • Connects many LANs using telephone lines, satelite, or fiber optic cables.

Example: The Internet is the largest WAN.

Network Components

A typical computer network consists of:

  1. Computers (nodes): Devices connected to the network. Example: Desktop, Laptop
  2. Network Interface Card (NIC): Enables computers to connect to the network. Example: Ethernet car
  3. Transmission Media: Pathway for data transmission. Example:Twisted pair, Fiber optic, Wi-Fi
  4. Switch/Hub: Connects multiple computers within a network. Example: Network switch
  5. Router: Connects different networks together (like LAN to Internet). Example: Internet router
  6. Modem: Converts digital signals to analog for transmission. Example: Home internet modem
  7. Server: Provides data or services. Example: File server, Print server.
  8. Client: Receives data or services. Example: Student's computer.

Functions of an information System

  • Input: Collecting raw data from users (e.g., typing marks into a computer).
  • Processing: Converting raw data into useful information (e.g., calculating totals).
  • Output: Producing reports or results (e.g., printing report cards).
  • Storage: Saving data and information for future use.
  • Feedback: Providing output that helps to improve future operations.

Type of Network Topologies

Network topology is the layout or arrangement of computers and cables in network.

Bus Topology

This is a network topology in which all computers are connected to one central cable (backbone).

Advantages

  • It is easy to install
  • Low cost

Disadvantages

  • If cable fails, whole network stops

Star Topology

This is a network topology in which all computer are connected to a central hub or switch

Advantages

  • Easy to manage
  • One cable failure doesn't affect others

Disadvantages

  • Requires more cables
  • Hub failure affects all

Ring Topology

This is a network topology in which each computer is connected to two others forming a ring

  • No collisions
  • Equal access

Disadvantages

  • Failure of one computer breaks the network

Mesh Topology

This is a network topology in which every computer is connected to every other.

Advantages

  • Reliable
  • No single point of failure

Disadvantages

  • Very expensive
  • Many cables

Tree Topology

This is a network topology that combines star and bus topologies

Advantages

  • Easy expansion

Disadvantages

  • Failure of backbone affects whole network

Network Devices and Their Functions

Hub: Connects multiple computers in a network and sends data to all devices.

Switch: Connects computer and sends data only to the intended device (faster than a hub)

Router: Connects different networks and directs data to the correct destination.

Bridge: Connects two LANs using the same protocol.

Gateway: Connects networks using different protocols.

Repeater: Boosts signals to allow longer cable connections.

Modem: Converts digital signals to analog and vice versa for internet communication.

Advantages of Computer Networks

  1. Resource sharing: Share printers, scanners, and files easily.
  2. Communication: Send emails, messages, or video calls easily.
  3. Data sharing: Users can access the same files or databases.
  4. Cost saving: Reduces hardware and software duplication.
  5. Remote access: Users can access system from anywhere.
  6. Backup and security: Easier to back up data from a central server.

Disadvantages of Computer Networks

  • Security risk: Hackers may access or steal data.
  • Virus spread: Viruses can spread quickly from one computer to another.
  • Network failure: If the network fails, users may lose access to data.
  • High setup cost: Requires cables, routers, switches, etc
  • Privacy issues: Personal information may be exposed
  • Requires skilled personnel: Needs network administrators and technicians.

Network Security Measures

To protect data and users, the following security measures are used:

  • Passwords: Limited access to authorized users.
  • Firewalls: Control incoming and outgoing network traffic.
  • Antivirus software: Detect and remove malicious programs
  • Encryption: Converts data into unreadable form for security.
  • Backup: Copy important data reqularly to prevent loss.
  • User access control: Assign access rights based on user role.

Client-Server Network

Definition

A Client-Server Network is a type of network where one powerful computer called the server provides resources or services to other computers called clients.

Explanation

In this type of network:

  • The server controls all network activities such as file storage, printing, and internet access.
  • The clients request for services from the server.
  • The server processes the requests and sends back results.

Examples

  • A school computer lab where student's computers connect to a main server.
  • Banking system where all teller's computers access a central database server.
  • Company networks where employees share files through one central file server.

Advantages of Client-Server Network

  • Centralized management: All files and data are store in one place
  • Data security: Server can control user access with passwords.
  • Data backup: Easy to back up data from one central location
  • Resource sharing: Many users can share printers, files, and software.

Disadvantages of Client-Server Network

  • Expensive: Requires powerful servers and software.
  • Requires skilled personnel: Needs a network administrator
  • Server failure: If the server stops working, the whole network is affected.
  • Maintenance cost: Servers require regular maintenance.

Peer-To-Peer (P2P) Network

Definition

A Peer-To-Peer Network a type of network where each computer acts both as a client and a server - meaning that all computers are equal and share resources directly with one another

Explanation

  • There is no central server.
  • Each computer (peer) can share files, printers, or internet connection with others.
  • Common in small networks such as homes or small offices.

Examples

  • File sharing between two laptops using a cable or Wi-Fi.
  • Bluetooth file transfer between phones.
  • Home networks where family computers share files.
  • P2P applications like BitTorrent.

Advantages of Peer-To-Peer Network

  • Easy to set up: No special server needed
  • Low cost: Cheap to install and maintain.
  • No central failure: If one computer fails, others still work.

Disadvantages

  • Poor security: Each computer control its own access.
  • Data duplication: Same files may exist on many computers.
  • Hard to manage: Difficult to control when many computers are connected
