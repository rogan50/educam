Information System

Meaning of an Information System

An Information System (IS) is a set of components(people, hardware, software, data, and procedures) that work together to collect, process, store, and distribute information for decision-making and control in an organization

Definition

An Information System is a system that collects data, processes it into useful information, and distributes it to peope who need it.

Explanation

Every organization - such as a school, hospital, or business - needs information to makes decisions.

An information system helps them to:

  • Gather raw data (like student marks or sales)
  • Process it (calculate totals or averages)
  • Produce useful information (reports, charts, etc.)

Example

In a school:

  • Raw data: Student's marks from exams
  • Processing: Calculating average and grades
  • Information: Producing a report card

That report card is the output of an information system

Component of an information system

An information system has five main components:

  • Hardware: Physical devices used to input, process, output and store data. Example: keyboard, computer, printer, hard disk
  • Software: programs and instructions that tell the hardware what to do. Example: Microsoft Excel
  • Data: Raw facts that are processed to produce information. Example: Marks, names, dates
  • Peopls: Users who operate the system and use the output. Example: Teachers, managers
  • Procedure: Steps or rules that guide how data is collected and processed.

All five components must work together for the information system to be effective.

Difference Between Data and Information

Data: Raw facts and figures.
Example: 60, 70, 80.

Information: Processed and meaningful data.
Example: Average score is 70%

Functions of an information System

  • Input: Collecting raw data from users (e.g., typing marks into a computer).
  • Processing: Converting raw data into useful information (e.g., calculating totals).
  • Output: Producing reports or results (e.g., printing report cards).
  • Storage: Saving data and information for future use.
  • Feedback: Providing output that helps to improve future operations.

Types of Information Systems

Information system are classified based on their purpose and users.

Transaction Processing System (TPS)

This is and information system that handles day-to-day business operations. It processes large amount of data quickly and accurately

Examples:

  • Supermarket billing system
  • Payroll system
  • Banking transaction system

Management Information System (MIS)

This is an information system that provides regular reports to middle managers.
It helps in planning, controlling, and monitoring performance.

Example: A school MIS may show attendance trends or exam results.

Decison Support System (DSS)

This is an information system that assists managers in making decisions using data and models. It analyzes "what if" Situations.

Example: A business checking "What will happen if prices increase by 10%"

Executive Information System (EIS)

This is an information system that provides summarized information for senior executives. It often uses graphs and charts for quick understanding.

Example: Company CEO checking overall sales performance by region.

Office Automation System (OAS)

This is an information system that helps in day-to-day office work like typing, emailing, and scheduling. It increases communication and efficiency.

Examples: Microsoft Word, Excel, Outlook, Google Workspace

Knowledge-Based System (KBS) or Expert System

This is an information system that uses Artificial Intelligence (AI) to ack like a human expert. It gives advice or diagnosis based on stored knownledge.

Examples: Medical diagnosis software, Weather forecasting systems

Importance of Information Systems

  • Faster decision making:Managers can get reports and make quick decisions.
  • Accuracy: Reduces human errors in calculations and reports.
  • Efficiency: Saves time and effort in handling large data.
  • Communication: Enables easy sharing of information.
  • Cost saving: Reduces need for paper and manual work.
  • Monitoring: Helps track performance and progress easily.

Disadvantages / Limitations

High cost: Expensive to buy and maintain systems.

Security risk: Data can be hacked or lost.

Requires skilled staff: Users must be trained to operate the system.

System failure: If computers crash, operations stop.

Privacy concerns: Sensitive data can be exposed.

Differences Between a Maual and Computerized Information System

  • Speed: Manual system is slow whereas Computerized system is fast
  • Accuracy: Manual system is prone to errors whereas Computerized is highly accurate
  • Storage: Manual system Uses paper files whereas Computerized uses electronic databases
  • Cost: Manual system less cost than computerized system
  • Security: Manual system is easy to lose or damage whereas Computerized system can be protected with passwords
  • Example: Manual system we have Handwritten receipts whereas for Computerized system We have Computerized billing system

Characteristics of a Good Information System

  • Accuracy: Information should be correct.
  • Timeliness: Information must be available when needed.
  • Completeness: Should contain all necessary details.
  • Relevance: Must be useful to the user.
  • Accessibility: Should be easy to access.
  • Security: Information must be protected from unauthorized access.
