Cloud Computing

Meaning of Cloud Computing

Cloud computing is the used of the Internet to store, manage, and process data instead of using a personal computer or local server.
In simple terms:
Cloud computing means keeping and using your files and programs online instead of on your own computer.

Example:

  • When you save your pictures or documents in Google Drive instead of your flash drive.
  • When you send an email using Gmail.
  • When you watch a movie on Netflix.

All these are examples of cloud computing.

Definition

Cloud computing is a technology that delivers computing services such as storage, software, databases, and servers over the internet.

It allows users to access these resources anytime and anywhere, using any internet-connected device.

Characteristics of cloud computing

  • On-demand self-service: You can use cloud services whenever you need them without human help.
  • Broad network access: You can access cloud services through the internet using phones, tablets, or computers
  • Resource pooling: Many users share the same cloud resources, but their data remains private.
  • Scalability: You can easily increase or reduce the amount of storage or power you need.
  • Reliability: Data us backed up and available most of the time.
  • Security (optional): Data is stored safely and protected by the service provider.

Types of Cloud Computing Services (service models)

There are three main types of cloud services commonly called the Cloud Models

Infrastructure as a Service (IaaS)

  • Provides basic computer resources such as servers, storage, and networking over the internet

Example: Amazon Web Services (AWS), Google Compute Engine.

Used by: Network engineers and system administrators.

Software developers.

  • Provides ready-made software online.
  • Users can access the software through a web browser without installing anything.

Example: Gmail, Google Docs, Dropbox, Zoom.

Used by: Students, offices, and business.

Types of Cloud Deployment Models

Public Cloud: Services are open to everyone on the internet. Example: Google Cloud, AWS

Private Cloud: Used by one organization only; more secure. Example: Banks, government offices

Hybrid Cloud: Combines both public and private clouds. Example: Large companies

Community Cloud: Shared by organizations with similar goals. Example: Schools, hospitals

Advantages of Cloud Computing

  • Cost effective: No need to buy expensive hardware or software.
  • Accessibility: You can access your data from anywhere with internet.
  • Automatic: Software updates happen automatically.
  • Data backup: Your data is store safely with backups.
  • Collaboration: Many users can work on the same file together.

Disadvantages of Cloud Computing

  • Needs Internet: Without Internet connection, you cannot access data.
  • Security risk: Data stored online can be hacked or stolen.
  • Limited control: You depend on the service provider for maintenance.
  • Downtime: If the provider's system fails, you may lose access temporarily.
  • Privacy issues: Some providers might access user's data.

Uses/Applications of Cloud Computing

  • File storage: Storing files and documents (e.g., Google Drive, Dropbox).
  • Email storage: Gmail, Outlook, Yahoo Mail.
  • Web hosting: Hosting websites and apps on cloud servers.
  • Education: Online learning platforms like Google Classroom or Coursera
  • Software development: Developers build and test applications using cloud tools.
  • Video streaming: YouTube and Netflix store and stream videos from cloud servers.
  • Backup and recovery: Automatic backup of important files and systems.

Security in Cloud Computing

  • Cloud providers use encryption to protect user's data
  • Users should use strong passwords and avoid sharing login details
  • Always log out after using cloud services.
