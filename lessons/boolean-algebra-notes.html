Boolean Algebra

1. What is Boolean Algebra?

Boolean Algebra is a branch of mathematics used in computers and digital electronics to work with values that are either TRUE or FALSE, represented as 1 and 0.

It is used to design logic gates, circuits, and decision-making in programming.

2. Boolean Values

  • 1 = TRUE (ON)
  • 0 = FALSE (OFF)

5. Boolean Laws and Rules

A. Identity Laws

  • A + 0 = A
  • A · 1 = A

B. Null Laws

  • A + 1 = 1
  • A · 0 = 0

C. Idempotent Laws

  • A + A = A
  • A · A = A

D. Inverse Laws

  • A + A' = 1
  • A · A' = 0

E. Commutative Laws

  • A + B = B + A
  • A · B = B · A

F. Associative Laws

  • (A + B) + C = A + (B + C)
  • (A · B) · C = A · (B · C)

G. Distributive Laws

  • A · (B + C) = A·B + A·C
  • A + (B · C) = (A + B)(A + C)

H. De Morgan’s Theorems

  • (A · B)' = A' + B'
  • (A + B)' = A' · B'

6. Boolean Expression Simplification

Simplification helps reduce logic circuits.

Example 1

Simplify: A + A·B

Solution: A(1 + B) = A·1 = A

Example 2

Simplify: (A + B)(A + B')

Solution: A + BB' = A + 0 = A

Example 3

Simplify: A'B + AB

This is XOR, so result = A ⊕ B

7. Logic Gates

Logic gates are electronic components that perform Boolean operations.

Types of Gates

  • AND Gate
  • OR Gate
  • NOT Gate
  • NAND Gate
  • NOR Gate
  • XOR Gate
  • XNOR Gate

8. Applications of Boolean Algebra

  • Design of digital circuits
  • Programming logic (if/else conditions)
  • Computer arithmetic
  • Search algorithms
  • Microprocessor design
