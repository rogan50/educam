Computer Architecture

Meaning of Computer Architecture

Computer Architecture refers to the design, structure, and organization of the main components of a computer system and how they work together to process data.

Definition

: Computer architecture is the blueprintor layout that shows how the CPU, memory, and input/output devices are connected and interact

Main Components of Computer Architecture

Computer architecture is mainly made up of three major components:

  1. Central Processing Unit (CPU)
  2. Memory Unit
  3. Input/Output (I/O) Devices

Let's understand each part clearly

Central Processing Unit (CPU)

The CPU called the brain of the computer. It performs all calculations, executes instructions, and controls all other parts of the computer.

Parts of the CPU

  • Arithmetic Logic Unit (ALU): Performs all arithmetic operations (add, subtract, etc.) and logical operations (AND, OR, NOT).
  • Control Unit (CU): Directs and controls all other units ot the computer. It tells them what to do and when.
  • Registers: Small, high-speed memory inside CPU that holds data temporarily during processing

Memory Unit

The memory unit stores, instructions, and results.
It is divided into main memory (primary) and secondary storage.

System Bus

A bus is a group of wires that connects all the main parts of the computer and allows them to communicate

types of bus

  • Data Bus: Transfers actual data between CPU, memory, and devices.
  • Address Bus: Carries memory addresses (where data should go).
  • Controls Bus: Sends control signals to coordinate all operations.

Von Neumann Architecture

Most modern computers use the Von Neumann Architecture, designed by John Von Neumann in 1945

Features

  • Both data and instructions are stored in the same memory.
  • The CPU fetches one instructions at a time (sequentially).
  • It uses a single path for data and instructions (called the system bus).

Main Components

  • Input Unit
  • Output Unit
  • Memory Unit
  • Control Unit
  • ALU (Arithmetic Logic Unit)

Functions of the Control Unit (CU)

  • Controls the flow of data between CPU and other parts.
  • Fetches instructions from memory.
  • Decodes instructions to understand what to do.
  • Executes commands in proper sequence.

Functions of the ALU (Arithmetic Logic Unit)

  • Performs arithmetic operations: addition, subtraction, multiplication, division.
  • Performs logical operations: AND, OR, NOT, comparison.
  • Store intermediate results in registers.

Functions of Register

Register are small storage areas insided the CPU that:

  • Stores data temporarily
  • Speed up processing
  • Hold instructions, memory addresses, or results.

Flow of Data in Computer Architecture

Here's how data flows when you perform a task on a computer:

  1. Input: User enters data using an input device.
  2. Storage: Data is stored temporarily in memory.
  3. Processing: CPU (ALU + CU) processes the data.
  4. Output: Results is display or printed.
  5. Storage (optional): Results may be saved permanently.

Differences Between Computer Architecture and Computer Organization

  • Meaning: Computer Architecture is the design and structure of computer components whereas Computer Organization is how components actually work and connect.
  • Focus: Computer Architecture is theoriticlly design whereas computer organization is practically implemented.
