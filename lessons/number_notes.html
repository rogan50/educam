Number Systems in Computer

1. What is a Number System?

A number system is a way of representing numbers using a set of digits. Every number system has digits and a base (radix).

2. Types of Number Systems

Number System Base Digits Used Example
Binary 2 0, 1 101101
Octal 8 0–7 725₈
Decimal 10 0–9 345
Hexadecimal 16 0–9, A–F A2F₁₆

3. Decimal Number System (Base 10)

The decimal system uses digits 0–9. It is the number system humans use every day.

Place values:

  • 10⁰ = 1 (Ones)
  • 10¹ = 10 (Tens)
  • 10² = 100 (Hundreds)
  • 10³ = 1000 (Thousands)

Example: 452 = 4×100 + 5×10 + 2×1 = 452

4. Binary Number System (Base 2)

Binary uses digits 0 and 1 only. It is the language of computers.

  • 1 bit = 0 or 1
  • 4 bits = Nibble
  • 8 bits = Byte

Place values: 2⁰, 2¹, 2², 2³, 2⁴, 2⁵...

Example: 1011₂ = 1×8 + 0×4 + 1×2 + 1×1 = 11₁₀

5. Octal Number System (Base 8)

Digits: 0–7

Example: 527₈ = 5×64 + 2×8 + 7×1 = 343₁₀

7. Conversion Between Number Systems

A. Binary to Decimal

Multiply each digit by 2ⁿ.

Example: 1101₂ = 13₁₀

B. Decimal to Binary

Use division by 2. Read remainders from bottom.

Example: 25₁₀ = 11001₂

C. Binary to Octal

Group binary digits in 3s.

Example: 110101₂ = 65₈

D. Binary to Hexadecimal

Group binary digits in 4s.

Example: 11100011₂ = E3₁₆

E. Decimal to Hexadecimal

Use division by 16.

Example: 75₁₀ = 4B₁₆

8. Why Computers Use Binary

  • Computers use electrical signals (ON = 1, OFF = 0)
  • Binary is reliable
  • Easy to implement in circuits
  • Reduces errors

11. Quick Memory Tricks

Hex ↔ Binary:

  • A = 1010
  • B = 1011
  • C = 1100
  • D = 1101
  • E = 1110
  • F = 1111

Powers of 2: 1, 2, 4, 8, 16, 32, 64, 128, 256…
